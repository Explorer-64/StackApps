## IDENTITY
name: PWA Toolkit
description: Scaffold production-ready Progressive Web Apps — generate manifest.json with correct icon entries, configurable service workers with caching strategies, and run installability audits against any live URL.
category: developer-tools
contact: support@pwatoolkit.dev

## SUMMARY
tagline: Ship installable PWAs faster — manifest, service worker, and compliance audits in one toolkit.
audience: Web developers building installable Progressive Web Apps who need correct manifests, service workers, and PWA compliance without manual configuration.
capabilities:
  - scaffold-manifest: Generate a valid manifest.json with icon size entries and theme configuration
  - generate-service-worker: Generate a service worker with cache-first, network-first, or SWR strategy
  - audit-pwa-requirements: Full installability audit against a live URL — manifest, HTTPS, icons, service worker
  - validate-icon-set: Check an icon ZIP or URL set for PWA size compliance and maskable padding
  - check-installability: Quick pass/fail installability check with specific remediation hints
