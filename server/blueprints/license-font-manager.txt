## IDENTITY
name: License Font Manager
description: Track webfont entitlements, subsetting plans, and embedding compliance for brand rollouts. Manages font licensing records — does not generate imagery or icon assets.
category: design
contact: registry@stackapps.app

## AUTH
provider: api-key
method: api-key

## ACCESS
preferred: mcp
last-resort: api

## CAPABILITY: inventory-faces
description: List all font families and faces active in the workspace with license tier and usage scope.
input: []
output:
  - type: json
    description: Array of font families with face names, foundry, license tier, and allowed use flags.
auth-required: true
scope: read-only

### MCP
tool: inventory_faces

## CAPABILITY: embedding-policy-check
description: Return allowed embedding flags for a font — web, app, desktop, and print permissions.
input:
  - name: font_family
    type: string
    required: true
    description: Font family name to look up, e.g. "Inter" or "Geist".
output:
  - type: json
    description: Embedding permission flags per use type with foundry name and license reference URL.
auth-required: true
scope: read-only

### MCP
tool: embedding_policy_check

## CAPABILITY: subset-recommendations
description: Suggest Unicode range subsets for a font to minimize web bundle size.
input:
  - name: font_family
    type: string
    required: true
    description: Font family to generate subset recommendations for.
  - name: target_scripts
    type: string
    required: false
    description: Comma-separated script targets, e.g. "latin,latin-ext". Default "latin".
output:
  - type: json
    description: Recommended unicode-range values per subset with estimated size savings in KB.
auth-required: true
scope: read-only

### MCP
tool: subset_recommendations
