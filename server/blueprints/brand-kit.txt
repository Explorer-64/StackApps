## IDENTITY
name: Brand Kit Studio
description: Build and export complete brand identity systems — color palettes, logo SVG variants, typography stacks, favicon packages, and brand bundles — from a brand name or hex seed color.
category: design
contact: support@brandkit.studio

## SUMMARY
tagline: Your entire brand identity from a name and a color.
audience: Indie developers, founders, and designers building consistent brand identities for web products and SaaS apps.
capabilities:
  - compose-color-palette: Accessible color system with primary, secondary, and neutral swatches
  - create-logo-variations: Light, dark, and monochrome SVG logo variants
  - generate-favicon: Favicon set (16, 32, 48px PNG) from an uploaded logo URL
  - draft-typography-stack: Heading and body font pairing with fallback stacks
  - design-app-icon: Single SVG app icon glyph for UI surfaces and app listings
  - export-brand-bundle: ZIP of all brand assets — logos, colors, fonts, and favicon set
