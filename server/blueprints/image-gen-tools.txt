## IDENTITY
name: Image Gen Tools
description: AI-powered text-to-image generation, upscaling, background removal, and style transfer at any resolution. Production-ready API built for developer scripts, CI pipelines, and agent workflows.
category: design
contact: support@imagegentools.dev

## SUMMARY
tagline: Text-to-image and image processing at any scale, via a developer-first MCP API.
audience: Developers and automation engineers who need programmatic image generation and processing without a design tool.
capabilities:
  - generate-image: Generate a single raster image from a text prompt at any resolution
  - upscale-image: Upscale an existing image 2x or 4x using AI super-resolution
  - remove-background: Remove the background from any image, returning a transparent PNG
  - apply-style-transfer: Apply an artistic style preset to an existing image
  - batch-process: Queue multiple generate or transform jobs and poll for results
